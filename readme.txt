Moneybird voor Wordpress is een plugin voor Wordpress wat het mogelijk maakt om je moneybird account te koppelen aan Wordpress.

== Addons ==

Moneybird User registration 0.1

- Registratie formulier voor klanten
- Klanten kunnen inloggen met eigen gebruikersnaam en wachtwoord
- Klanten kunnen hun eigen facturen inzien

http://moneybird.csorbamedia.com/addons/

== Omschrijving ==

Met Moneybird voor Wordpress kan je klanten en facturen direct onderhouden in Wordpress.

Mogelijkheden in versie 0.1:

- Dashboard met laatste 10 klanten en facturen
- Aanmaken klanten, wijzigen klanten
- Aanmaken concept factuur per klant.
- Maken van definitieve factuur, versturen van reminders.
- Verwijderen van facturen en klanten
